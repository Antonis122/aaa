How to setup Git and link GitHub account

How to install Git and GitHub

  1. Go to their website https://git-scm.com/install/
  2. Download and Install Git based on your operating system.
  3. Create a new folder, right click inside the new folder and click open Git Bash here. In Windows 11, right click inside the folder then click "show more options".
  4. Git is successfully installed

How to check if Git is installed properly in VS code

  1. Click on the directory and type cmd. It should open a terminal.
  2. Inside the terminal, type code .
  3. It should open VStudio, hold Ctrl + ` to open the terminal.
  4. Click the arrow down in your terminal and click Git bash to open the git terminal.
  5. Use the terminal to link your Github

How to link VS code to GitHub

  1. Use Git inside VS code and type the following
    • git config --global user.name "Your Name"
    • git config --global user.email "your email address used in your GitHub account"
    • git config --global push.default matching
    • git config --global alias.co checkout
    • git init
  2. Go to github then create a new repository and type your desired name. Make sure it is set to public.
  3. Copy the text seen on your page and paste inside VS code git, Refer to command below :
    • git init
    • git add README.md
    • git commit -m "first commit"
    • git branch -M main
    • git remote add origin (your own link)
    • git push -u origin main
  4. Then click the profile in the bottom left
  5. Login your github account
  6. Then click automatically save changes and Turn on Cloud Changes then you're done!

How to import GitHub repository from other pc

  1. Create new folder, open git then input the following
    • git clone "link of your repository"
  2. It should import the github files inside the desired folder then you're done!

How to sync/upload saved file to GitHub?

  1. Install Conventional Commits from extensions
  2. After editing your code, go to source control from left side then hover over the changes
  3. Click the + then click the little circle above commit
  4. After that, click feat and choose from the options on what you want the description to be.
  5. Lastly, click commit to sync the file to your github repository.
  6. Go to your github repository to check if it updated and then you're done!
